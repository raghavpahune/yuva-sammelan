# 🚩 युवा सम्मेलन — Yuva Sammelan 2026

> **युवा जागेगा – भारत आगे बढ़ेगा!**

Landing page for **Yuva Sammelan**, organized by **Rashtriya Swayamsevak Sangh (RSS), Kharbi Nagar, Nagpur** — celebrating the Shatabdi Varsh (1925–2025).

## 📋 Event Details

| | |
|---|---|
| **📅 Date** | 5 July 2026, Sunday |
| **🕔 Time** | 5:00 PM |
| **📍 Venue** | Shri Samarth Lawn, Kharbi Nagar, Nagpur |
| **👥 Age Group** | 16 – 35 years |

## ✨ Features

- Anti-gravity / levitation themed design with floating particles
- UPI payment integration (Google Pay, PhonePe, Paytm, BHIM)
- Dynamic amount input — all payment links update in real-time
- Copy UPI ID with toast confirmation
- WhatsApp share button
- Mobile-first responsive design
- Zero dependencies — pure HTML + CSS + JS

## 🚀 Usage

Just open `index.html` in a browser. No build step, no server needed.

Or host it via [GitHub Pages](https://pages.github.com/):
**Settings → Pages → Source: main / root**
